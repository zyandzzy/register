# 任务管理 UI 设计说明

## 新版 UI 特点

### 1. 现代化视觉设计

**渐变背景**
- 使用紫色到粉色的渐变背景
- 营造现代、专业的视觉效果
- 支持暗色模式自动切换

**卡片式布局**
- 替代传统表格
- 每个任务都是独立的卡片
- 带阴影和悬浮效果

### 2. 清晰的层级展示

**根任务**
- 蓝色左边框（5px）
- 📁 文件夹图标
- 大字号标题（20px）
- 白色背景卡片
- 显示子任务数量

**子任务**
- 绿色连接线和圆点
- 📄 文档图标
- 嵌套在父任务下方
- 独立的渐变背景
- 视觉上明确的层级关系

**连接线设计**
- 垂直绿色实线连接
- 圆点标记子任务位置
- 最后一个子任务后不显示线

### 3. 优化的信息展示

**优先级图标**
- 🔴 紧急
- 🟠 高
- 🟡 中
- 🔵 低

**状态标签**
- 待处理：灰色
- 进行中：橙色
- 已完成：绿色（整个卡片带绿色渐变）
- 已取消：红色

**元数据**
- 卡片式显示
- 浅灰色背景
- 包含：优先级、截止时间、子任务数

**任务描述**
- 独立的灰色区域
- 圆角边框
- 清晰可读的行高

### 4. 交互体验

**悬浮效果**
- 鼠标悬停时卡片上浮
- 阴影加深
- 子任务卡片向右移动

**按钮设计**
- 圆形按钮简洁美观
- 图标一目了然
- 颜色编码：
  - 绿色：添加子任务
  - 蓝色：编辑
  - 红色：删除

**动画效果**
- 淡入动画（fadeIn）
- 平滑过渡
- 响应式变化

### 5. 布局结构

```
┌─────────────────────────────────────────┐
│  📁 任务管理              [创建任务]     │  ← 头部
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│ 📁 根任务标题    [✅ 状态]              │
│   ⚡ 优先级 | 📅 截止 | 👥 子任务: 2个   │  ← 根任务卡片
│   任务描述...                            │
│                                          │
│   子任务列表                             │
│   ┌────────────────────────────────┐   │
│   ● ─ 📄 子任务1  [✅]  [编辑] [删除] │  ← 子任务1
│   │   描述... 📅 截止时间            │
│   └────────────────────────────────┘   │
│   ┌────────────────────────────────┐   │
│   ● ─ 📄 子任务2  [✅]  [编辑] [删除] │  ← 子任务2
│       描述... 📅 截止时间            │
│   └────────────────────────────────┘   │
└─────────────────────────────────────────┘
    [➕] [✏️] [🗑️]  ← 操作按钮
```

## 对比旧版改进

### 旧版（表格）
- ❌ 难以区分层级
- ❌ 信息密集拥挤
- ❌ 缺少视觉吸引力
- ❌ 子任务只有简单标记

### 新版（卡片）
- ✅ 层级一目了然
- ✅ 信息分组清晰
- ✅ 现代美观设计
- ✅ 树形结构可视化

## 颜色方案

**主色调**
- 蓝色（#409EFF）- 根任务
- 绿色（#67C23A）- 子任务/已完成
- 紫色渐变 - 背景

**状态颜色**
- 灰色（#909399）- 待处理
- 橙色（#E6A23C）- 进行中
- 绿色（#67C23A）- 已完成
- 红色（#F56C6C）- 已取消

## 响应式设计

- 大屏：卡片宽度自适应
- 中屏：保持完整布局
- 小屏：卡片堆叠显示
- 移动端：优化触摸操作

## 性能优化

- computed 计算树形结构
- CSS动画硬件加速
- 最小化重绘
- 虚拟滚动（可扩展）
